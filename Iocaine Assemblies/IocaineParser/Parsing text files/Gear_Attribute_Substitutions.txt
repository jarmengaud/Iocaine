// How to add substitutions.
// Generally, just follow the format below.
// Lines starting with *:: are substitutions that are performed on every item.
// Try to be sparing on these as it will slow down processing much more dramatically
// than the per-item substitutions.
// The per-item substitutions start with the item name (as shown in inventory, not the full name).
// The :: after the name is just a seperator, followed by the text to be replaced.
// The => is another seperator, followed by the text to be filled into the item.
// This text will then be parsed in the "Gear_Attribute_Filters.txt" file.
// Basically, this is done to either:
// 1. Enter a quantity where none exists in the description, but has been tried and tested.
// 2. To standardize the naming where different description text means the same thing.
// An example of both 1 and 2 would be:
//   Entois trousers::Adds "Recycle" effect=>"Recycle"+3
//   Sct. Beret +2::"Recycle"=>"Recycle"+25

//   Neither item quantifies the value, so we add it here.
//   Sct. Beret +2 just says "Recycle" instead of 'Adds "Recycle" effect'
//   We fix this by replacing 'Addes "Recycle" effect' with "Recycle"+3
//  		 and by replacing "Recycle" with "Recycle"+25.
//	 This makes it so that we only have to parse for "Recycle"+(value).
//
//   Blank lines or lines starting with // are ignored.

// Elemental symbol codes
*::\uE000=>Fire
*::\uE001=>Ice
*::\uE002=>Wind
*::\uE003=>Earth
*::\uE004=>Lightning
*::\uE005=>Water
*::\uE006=>Light
*::\uE007=>Dark

// Quantify class refresh/regen with values of 1
*::Adds "Refresh" effect=>"Refresh"+1
*::Adds "Regen" effect=>"Regen"+1
*::(?:Decreases|Reduces) movement speed=>Movement speed -12
